# Parjai Web

Site vitrine Next.js pour le clan mandalorien **Parjai** (serveur RP Star Wars).

## Stack
- Next.js 16 (App Router) + TypeScript + Tailwind CSS v4

## Structure
```
src/
  app/           # Pages (accueil, lore, membres, règles, recrutement)
  components/    # Composants partagés (Navbar, Footer)
```

## Thème
- Palette sombre : fond noir/gris foncé, accents dorés (#c9a84c)
- Variables CSS définies dans `globals.css`
- Inspiration visuelle : beskar, armures mandaloriennes

## Commandes
- `npm run dev` — serveur de développement
- `npm run build` — build de production
- `npm run lint` — linting ESLint

## Conventions
- Langue du site : français
- Contenu RP/fictif basé sur l'univers Star Wars (Mandalorien)
- Termes Mando'a utilisés quand approprié
